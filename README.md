# Real-Estate-Website
---Live-Link:  [Alex Estate](https://real-estate-e912d.firebaseapp.com/)

# Features
 - Google, Github Secure Login
 - Custom registration
 - Save Favorite Estates
 - Eye catching slider for properties
 - Maps for individual estates
